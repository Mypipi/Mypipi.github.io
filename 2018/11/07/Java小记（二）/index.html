11.构造方法是类的一种特殊方法。
作用：完成对新对象的初始化
特点:
1.方法名与类名相同
2.没有返回值
3.创建一个类的新对象的时候，系统会自动调用该类的构造方法完成对新对象的初始化
4.一个类可以有多个构造方法
5.每个类都有个默认构造方法
主要用处：初始化你的成员变量
