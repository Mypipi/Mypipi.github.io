关于Java基础的复习（十）

1.多态

①定义：
父类型/父接口类型 变量 = new 子类对象();
变量.方法 调用的是子类的重写方法
②多态中成员的特点
编译看父类,运行看子类.
③多态的转型
1)Java中强制数据类型的语言,参数运算数据要保证数据类型统一
Fu f = new Zi();
f:两边的数据类型不一样,发生自动提升类型,向上转型.子类(小)转化为父类(大)
2)好处:可以调用子类和父类的公有成员
坏处:无法调用子类的特有成员
如果要调用子类的特有成员要进行强制转换
数据类型 变量 = (数据类型)变量—由子类转为父类

2.封装,继承,多态的作用

①封装:把对象的属性,方法实现的细节隐藏,仅仅对外提供一些公共访问方式.
②继承:子类自动拥有父类所有可以继承的属性和方法.
③多态:配合继承与方法重写提高了代码的复用性与扩展性,如果没有方法重写,多态并无意义.
