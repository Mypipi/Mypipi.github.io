5.代码块

①构造代码块(new 一次执行一次,优先于构造方法)
②静态代码块(只执行一次)
③构造方法(new一次执行一次)
优先顺序:②->①->③

6.Java的修饰符

①public权限修饰符,公共访问,类,方法,成员变量
②protected权限修饰符,方法,成员变量
③默认,类,方法,成员变量
④private,私有访问,方法,成员变量
⑤static,方法,成员变量
⑥final,类,方法,成员变量,局部变量
⑦abstract,类,方法

7.抽象类,接口与方法

①抽象类作为方法的参数,只能传递其子类对象,这也是多态的体现
②抽象类作为方法的返回值,返回值是其子类对象
③接口作为方法的参数必须传递其实现类对象
④接口作为方法的返回值,返回其实现类对象
